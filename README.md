# GraphsMinimalMatrixCode
**Программа осуществляет чтение графов из файла**

Формат графов может быть *graph6* или *digraph6*. 
При этом оба формата могут идти в перемешку в одном файле.
Результатом работы программы является файл, в котором для каждого графа написан его матричный код и код графа в соответствующем формате. 

## Требование
**.NET 8.0** или выше, хотя возможно его можно в файле MinimalMatrixCode\MinimalMatrixCode.csproj поменять и будет работать _(не проверял ниже .NET 8.0)_

## Использование
в каталоге .\MinimalMatrixCode:
    **dotnet run <путь к файлу>**

в результате в этой же папке создастся файл **<путь к файлу>_result.txt**
    Например:\
    ( исходный код графа | его минимальный матричный код | код графа соответствующей перестановки вершин )\
    E?B_ | 11 | E??w\
    E?Bo | 75 | E?@w\
    G?r@`_ | 6402816 | G?KuE?
